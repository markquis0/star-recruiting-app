# Favicon Image

Please place the favicon image file here with the filename:
`favicon.png`

The favicon should be:
- A magnifying glass icon with a human head silhouette inside
- Pinkish-red icon elements (#F03A6F) on a dark purple background (#3F1E6F)
- Recommended size: 32x32 pixels or 64x64 pixels (will be scaled automatically)
- PNG format for best quality

The favicon will automatically be used in:
- Browser tabs
- Bookmarks
- Mobile home screen icons (when saved)
